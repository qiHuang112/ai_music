# UI Lane 知识库

这个目录给 UI lane 沉淀 AI Music 页面体验、截图观察、样式问题、验收方法和可复用设计结论。

## 当前职责

- 先熟悉 APP 的主要功能页面和当前 UI 状态。
- 在 product 明确要求前，不加入自动化 review 流程。
- 不直接修改业务代码，不直接提交或推送。
- 需要体验包时，先请求 product/architect 协调 android、ohos 或 ios lane 安装。

## 适合记录的内容

- 页面名称、入口路径、主要用户动作。
- 截图观察到的信息层级、布局、交互状态、异常或不一致。
- UI 优化建议及其影响范围。
- UI 验收需要的设备、包版本、截图列表和检查项。
- 相关 request、commit、thread 和验证日期。

## 不记录的内容

- 完整聊天原文。
- 账号、证书、设备隐私、token 或其它敏感信息。
- 一次性截图文件的原始大段路径列表；只记录结论和必要定位信息。
